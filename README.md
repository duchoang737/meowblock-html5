# Flower Block HTML5

Game logic Shikaku gồm 100 màn, được xây dựng từ tài liệu reverse-design và FTUE.

## Chơi

Mở GitHub Pages của repository này để chơi trực tiếp trên trình duyệt.

## Tính năng

- 100 màn tăng dần độ khó
- Hướng dẫn 4 bước
- Shape Constraint mở từ màn 9
- Sealed Number mở từ màn 14
- Mystery mở từ màn 19
- Gợi ý và lưu tiến độ trên thiết bị

Mã nguồn React/Vite nằm trong thư mục `source/`.
